# Lab 7 - Logistic Regression

This lab demonstrates the use of Logistic Regression to build a classification model that predicts whether a user will click on an online advertisement. The model is trained using features related to user behavior and demographics, such as age, income level, and internet activity.

The dataset used in this lab is **advertising.csv**, which contains information about users and their interaction with advertisements.

## Workflow:

- Importing and inspecting the dataset  
- Performing exploratory data analysis (EDA)  
- Visualizing relationships between variables using seaborn  
- Preparing the data and splitting it into training and testing sets  
- Training a Logistic Regression model  
- Generating predictions on the test data  
- Evaluating the model performance using a classification report  

## Files:

- `Lab7.ipynb` – Jupyter Notebook containing all steps and code  
- `advertising.csv` – Dataset used for training and testing  

## Results:

The trained model achieved an accuracy of approximately **91%**, indicating strong performance in predicting user interactions with advertisements.

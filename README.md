# ARTI308 - Machine Learning Labs

This repository contains all lab assignments and practical work completed for the Machine Learning course (ARTI308) at Imam Abdulrahman Bin Faisal University.

It demonstrates hands-on experience in data analysis, model development, and the application of core machine learning concepts using real-world datasets.

The work reflects a strong focus on writing clean, well-structured code and building a solid foundation for future development in machine learning and software engineering.

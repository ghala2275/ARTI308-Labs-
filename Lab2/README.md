# ARTI308-Lab2-Salary-Prediction

## Learning Goals

### 1. Identifying the Machine Learning Problem Type
 This project tackles a **Regression** problem under **Supervised Learning**.

**Why Regression?**
- The target variable (Salary) is a numeric, continuous variable.
- It needs to predict a value, rather than a class or a group.
- This formulation closely mirrors actual use cases of salary prediction in the real world.


### 2. Dataset Selection
The dataset used in this Assignment is taken from **Kaggle**.

**Dataset Name:** Salary Prediction Dataset  
**Description:** Dataset for salary prediction based on experience and education.

Employee demographic and professional information for machine learning tasks exist in the dataset. 


### 3. Target Variable
Target Variable. Target variable for this supervised learning task can be defined as:

- **Salary:** Annual salary of the employee in dollars.


### 4. Problem Statement
This project aims to create a machine learning model that can estimate the annual salary of an employee based on demographic and profession-related features. These factors, from age, gender, educational qualification, job title, and years worked are some well-understood pay factors in real life employment and impact their level.

We then formulated our problem as a **supervised regression task** task for the target variable (salary) that is a continuous numerical value and past labeled data. The model aims to learn the relationship between these input features and salary and to predict salaries of new and unknown employee records.

 It exemplifies the application of regression methods to real-world prediction tasks incorporating employee information. Furthermore, hands-on experience with data exploration, feature understanding, and a full machine learning workflow is obtained.

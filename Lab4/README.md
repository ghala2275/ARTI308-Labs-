# Lab 4 - Data Quality Assessment & Preprocessing

This repository presents my work for Lab 4, where I performed data quality assessment and preprocessing on the Heart Disease dataset.

The main steps included:

* Examining the dataset for quality issues
* Simulating and handling missing values using the Remove Records approach
* Identifying and removing outliers using the IQR technique
* Normalizing numerical features using both Min-Max scaling and Z-score standardization
* Analyzing feature correlation to determine whether PCA should be applied

Based on the analysis, PCA was not used due to the weak correlation between the selected features.

Dataset: Heart Disease Dataset
